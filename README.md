# Color Generator

A simple JavaScript project that generates random background colors and displays the HEX color code.

## Features

- Random color generation
- Dynamic background update
- HEX color display
- Interactive button

## Technologies Used

- HTML5
- CSS3
- JavaScript

## JavaScript Concepts Used

- DOM Manipulation
- Event Listeners
- Math.random()
- Math.floor()
- toString(16)
- textContent
- style.background
